# Roadmap

## v0.1 - Reference classification

- Build a small reference dataset for Saudi listed companies
- Classify companies by business class
- Link companies to Vision 2030 themes
- Generate a basic market overview report

## v0.2 - Full market coverage

- Expand the dataset to all Saudi Exchange listed companies
- Separate Main Market and Nomu
- Add sector and industry validation
- Add company profile fields

## v0.3 - Market metrics

- Add market cap classification
- Add liquidity classification
- Add trading volume bands
- Add price and performance fields

## v0.4 - Fundamental profile

- Add revenue, net income, margin, and ROE fields
- Add dividend profile classification
- Add debt and leverage indicators
- Add company quality tags

## v0.5 - Reports and dashboards

- Generate sector reports
- Generate company profile pages
- Add Streamlit dashboard
- Add searchable market map

## Future ideas

- Saudi market screener
- Sector comparison engine
- Vision 2030 exposure map
- AI assistant for Saudi listed companies
